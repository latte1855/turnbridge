# ✅ **F0501 — 平台存證作廢發票訊息**

來源：電子發票 Turnkey 系統需求規格書

---

# **📌 1. 訊息用途**

> **作廢發票（賣方→平台）之存證訊息。**
> 包含原發票號碼、買賣方統編、作廢日期、作廢原因等必要欄位。

---

# **📌 2. 訊息樹狀結構（依文件）**

```
CancelInvoice
 └── CancelInvoiceNumber        (作廢發票號碼)
 ├── InvoiceDate                (發票日期)
 ├── BuyerId                    (買方統一編號)
 ├── SellerId                   (賣方統一編號)
 ├── CancelDate                 (作廢日期)
 ├── CancelTime                 (作廢時間)
 ├── CancelReason               (作廢原因)
 ├── ReturnTaxDocumentNumber    (專案作廢核准文號)
 └── Remark                     (備註)
```

# ✅ **F0701 — 平台存證註銷發票訊息（Void Invoice）**

來源：電子發票 Turnkey 系統需求規格書

---

# **📌 1. 訊息用途**

> 此訊息用於「註銷（Void）」原本已開立並上傳平台存證的發票。
> 適用於 B2C / B2S 之「註銷」情境（非作廢）。

關鍵差別：

* **作廢 = Cancel（F0501）** → 發票當期或未使用
* **註銷 = Void（F0701）** → 發票已上傳，但因特殊原因需「銷案」

---

# **📌 2. 樹狀結構（依 MIG 原始格式）**

```
VoidInvoice
 ├── VoidInvoiceNumber         (註銷發票號碼)
 ├── InvoiceDate               (發票日期)
 ├── BuyerId                   (買方統一編號)
 ├── SellerId                  (賣方統一編號)
 ├── VoidDate                  (註銷日期)
 ├── VoidTime                  (註銷時間)
 ├── VoidReason                (註銷原因)
 └── Remark                    (備註)
```
